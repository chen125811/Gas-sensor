# Gas-sensor
# MOF Array Gas Sensing with Machine Learning

## Overview
This repository contains the code for machine-learning-assisted analysis of MOF array gas sensor data, including gas classification and concentration regression for single gases and binary gas mixtures.

## Background
This project is related to our research on MOF-based sensor arrays for complex gas analysis.  
The code is designed to process sensor response data, build machine learning models, and reproduce the main results reported in the corresponding paper/thesis.
